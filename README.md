# CrackAnalytics by Mahendra Singh — Deploy Instructions

SINGLE-PAGE website: 13 tabs, 430+ questions with answers, Resource Hub (56 files).

## Vercel pe live karne ke 3 steps:
1. https://vercel.com pe free account banao (GitHub se sign up)
2. "Add New Project" → is pure folder ko DRAG & DROP kar do
3. Site live!

## IMPORTANT — Deploy se pehle:
script.js kholo aur WORKSPACE = 'crackanalytics-site' ko apna unique
naam do (jaise 'crackanalytics-mahendra-2026') — visitor counter ke liye.

## Naye features:
- Har topic (Power BI, Tableau, Excel, SQL) ke questions Easy/Medium/Hard
  groups mein organized hain
- Data Science & AI tab — Python, Statistics, ML, GenAI questions
- Company-wise tab — company ke button pe click karo, uske questions khulte hain
- Resource Hub — SAB KUCH view-only hai: har PDF site ke built-in reader
  mein khulta hai, koi download button nahi (PPT/docx bhi PDF bana diye)

## PDF "view-only" ke baare mein SACH:
Question PDFs built-in reader mein khulte hain aur download button nahi
dikhta — casual users download nahi karenge. LEKIN technically koi bhi
web content 100% download-proof nahi hota (browser tools se file mil
sakti hai). Ye protection "discourage" karta hai, "impossible" nahi banata.

## File size:
Total ~80MB (Vercel free limit 100MB ke andar). Naye PDFs resources/
ke category folders mein daalo — lekin total 100MB ke andar rakho.
Bade files (Agile.pdf, Business Analyst Questions.pdf, ML ke bade PDFs)
isliye include nahi kiye — unhe Google Drive pe daal ke link karo.

## Google pe dikhne ke liye:
search.google.com/search-console pe site add karo, sitemap.xml submit karo.
